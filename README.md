**Smart Clipboard Manager for Windows**

Smart Clipboard Pro — это лёгкий и удобный менеджер буфера обмена для Windows, который автоматически сохраняет всё, что вы копируете: текст и изображения.

Больше никаких потерянных данных.
Один раз скопировал — всегда можешь вернуть.

**Возможности**

Автоматическое сохранение текста

Автоматическое сохранение изображений и скриншотов

Быстрый поиск по истории

Горячая клавиша Ctrl + Shift + V

Хранение последних 30 элементов

Удаление отдельных элементов

Полная очистка истории

Локальное хранение (без интернета)

**Как работает**

Программа работает в фоне и отслеживает буфер обмена Windows.

Когда вы копируете:

текст — он сохраняется в базе данных

изображение — оно сохраняется как файл и добавляется в историю

Открыть окно можно сочетанием клавиш:

Ctrl + Shift + V
**Требования**

Windows 10 / 11

Python (если запускать из исходников)

Готово ✅

**Где хранятся данные**

Все данные сохраняются локально:

AppData/Roaming/SmartClipboard

Никакой передачи данных в интернет.

**Безопасность**

Программа не отправляет данные в интернет

Всё хранится только на вашем компьютере

Нет рекламы и трекеров
